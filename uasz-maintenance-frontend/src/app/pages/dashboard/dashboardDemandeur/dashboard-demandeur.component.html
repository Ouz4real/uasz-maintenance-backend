Aide & support

help_outline

Questions fréquentes

Quelques réponses rapides avant de contacter le support.

Comment créer une nouvelle demande ?

Depuis votre tableau de bord ou l’onglet “Mes demandes”, cliquez sur “Nouvelle demande de panne”, renseignez les champs obligatoires (titre, lieu, type d’équipement, urgence) puis ajoutez une image de l’équipement (obligatoire). Le bouton “Enregistrer la demande” s’active uniquement si tout est valide.

Comment suivre l’avancement d’une demande ?

Le statut de chaque demande (En attente, En cours, Résolue) est visible dans vos listes. Cliquez sur une demande pour voir le détail (infos + image) et, si besoin, la relancer lorsqu’elle est bloquée trop longtemps.

Comment relancer une demande en attente ?

Ouvrez l’onglet “Mes demandes”, cliquez sur la demande concernée (statut En attente), puis utilisez le bouton “Relancer cette demande” dans la fenêtre de détails. (Le bouton devient actif après une certaine durée d’attente, par exemple 5 jours.)

Quand utiliser l’urgence “Haute” ?

Utilisez l’urgence Haute uniquement si la panne bloque un cours, un examen ou une activité critique (salle de TP, serveur pédagogique, etc.). Sinon, privilégiez Basse ou Moyenne.

Et si mon équipement n’est pas dans la liste ?

Sélectionnez “Autre” puis précisez le nom de l’équipement. Exemple : “AUTRE: Robinet – Couloir C03”. Votre demande reste valide.

Pourquoi mon image est refusée ?

Certaines images peuvent être trop volumineuses. Si un message en rouge s’affiche sous le champ “Image de l’équipement”, choisissez une image plus légère (ou compressez-la) puis réessayez.

Besoin d’un coup de main ?

Si vous ne trouvez pas la réponse à votre question, vous pouvez contacter l’équipe maintenance ou créer directement une nouvelle demande.
